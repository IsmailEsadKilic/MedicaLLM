# MedicaLLM Backend

MedicaLLM is a Medical AI Assistant backend built with FastAPI and LangChain.

## Requirements

- Python >= 3.13

## Setup

```bash
pip install -e .
```

OR

```bash
pip install -r requirements.txt
```

OR

```bash
uv pip install -e .
```

OR

```bash
uv pip install -r requirements.txt
```

## Running

```bash
python run.py
```
